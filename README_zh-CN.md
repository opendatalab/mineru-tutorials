# MinerU 实战训练营

[English](./README.md) | 简体中文

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/nanhongchuan/mineru-tutorials)

欢迎加入我们的 [训练营微信群](https://mineru.net/community-portal/?aliasId=794b55ee) 交流学习。

本仓库整理了 MinerU 实战训练营的课程材料，覆盖 MinerU 产品选型、本地与容器化部署、在线 API 调用、模型微调、OpenClaw Skill 开发、AI Agent 搭建，以及基于 OmniDocBench、Dingo、WebMainBench 的文档解析质量评测。

课程材料包括：

- 课程文档：Markdown 格式，便于在线阅读、检索和维护
- 课程课件：PDF 格式，便于预览和分享
- 课程视频：B 站合集链接，便于在线播放

## 适合人群

- 希望快速了解 MinerU 产品矩阵和使用方式的开发者
- 需要在本地、Docker 或国产算力环境中部署 MinerU 的工程人员
- 希望通过 MinerU Open API 构建文档解析、知识库、信息抽取应用的开发者
- 关注文档解析模型微调、质量评测和工程落地的算法与平台团队
- 希望借助 AI 编程工具完成 Agent、Skill 或科研工具开发的实践者

## 学习路径

建议按课程顺序学习：

1. 先通过第 01 课了解 MinerU 产品矩阵和场景选型。
2. 再通过第 02 课完成 MinerU 与 MinerU-HTML 的部署实践。
3. 如果希望免部署接入能力，学习第 03 课的在线 API。
4. 如果需要模型定制，学习第 04 课的 MinerU 1.2B 微调流程。
5. 如果关注生态集成，学习第 05、06 课的 Skill 和 Agent 实践。
6. 第 07、08 课为选修内容，适合有质量评估、模型对标或模型优化需求的同学学习。
